1. Regresi Linier
Dalam tugas ini, dilakukan proses analisis data menggunakan regresi linier. Pertama, dataset dibaca dan dipisahkan menjadi variabel independen (X) dan dependen (y). Nilai yang hilang diolah dengan metode imputasi rata-rata, kemudian data kategorikal dikonversi menjadi bentuk numerik menggunakan encoding. Data selanjutnya dibagi menjadi data latih dan uji, lalu dilakukan standarisasi fitur agar memiliki skala yang sama. Model regresi linier kemudian dilatih menggunakan data latih dan diuji dengan data uji.
